## Required Images for the Website

Please create the following folder structure and add these images:

```
public/
└── images/
    ├── logo/
    │   └── logo.png           # Your store logo (recommended size: 200x50px)
    │
    ├── hero/
    │   ├── hero-1.jpg        # Main hero banner (recommended size: 1920x800px)
    │   └── hero-2.jpg        # Secondary hero banner (if needed)
    │
    ├── banners/
    │   ├── banner-1.jpg      # Collection banner (recommended size: 1200x400px)
    │   ├── banner-2.jpg      # Promotion banner
    │   └── banner-3.jpg      # Additional banner
    │
    └── payment/
        ├── visa.svg          # Visa logo
        ├── mastercard.svg    # Mastercard logo
        ├── amex.svg          # American Express logo
        ├── klarna.svg        # Klarna logo
        ├── applepay.svg      # Apple Pay logo
        ├── googlepay.svg     # Google Pay logo
        └── paypal.svg        # PayPal logo
```

### Image Requirements:

1. **Logo**
   - Format: PNG with transparent background
   - Size: 200x50px (or your preferred size)

2. **Hero Banners**
   - Format: JPG/JPEG
   - Size: 1920x800px
   - High quality, optimized for web

3. **Collection/Promotional Banners**
   - Format: JPG/JPEG
   - Size: 1200x400px
   - Should match your brand colors

4. **Payment Method Logos**
   - Format: SVG preferred (for better scaling)
   - Size: Will automatically scale
   - Official logos from payment providers

Please provide these images with the exact filenames mentioned above. Once you have them ready, I'll update the website to use them.
